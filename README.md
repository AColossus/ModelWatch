# ModelWatch

## ModelWatch是一个基于WebGL实现三维模型展示系统的网站
本系统主要实现的功能有：<br>
>1.模型的导入与展示。<br>
>2.模型项目的管理。<br>
>3.模型的创建与动画的简单设计。<br>
>4.模型的分享与下载。<br>
